### Task 3: Shift+Tab in PersonNode edit mode

**Files:**
- Modify: `web/src/components/PersonNode.tsx` — add reverse tab cycling

- [ ] **Step 1: Read PersonNode.tsx tab handling**

Read `web/src/components/PersonNode.tsx` — find the Tab key handler (around lines 90-109). It currently cycles forward: name → role → team → name.

- [ ] **Step 2: Add Shift+Tab reverse cycling**

In the Tab key handler, check `e.shiftKey`. If true, cycle backwards:
- name → team (reverse)
- team → role (reverse)
- role → name (reverse)

```typescript
if (e.key === 'Tab') {
  e.preventDefault()
  if (e.shiftKey) {
    // Reverse cycle
    if (document.activeElement === nameRef.current) teamRef.current?.focus()
    else if (document.activeElement === roleRef.current) nameRef.current?.focus()
    else if (document.activeElement === teamRef.current) roleRef.current?.focus()
    else nameRef.current?.focus()
  } else {
    // Forward cycle (existing logic)
    if (document.activeElement === nameRef.current) roleRef.current?.focus()
    else if (document.activeElement === roleRef.current) teamRef.current?.focus()
    else if (document.activeElement === teamRef.current) nameRef.current?.focus()
    else nameRef.current?.focus()
  }
}
```

### Task 4: Enable KeyboardSensor for dnd-kit

**Files:**
- Modify: `web/src/hooks/useChartLayout.ts` — add KeyboardSensor

- [ ] **Step 1: Read useChartLayout.ts sensor setup**

Read lines 1-21 of `web/src/hooks/useChartLayout.ts`. Currently only `MouseSensor` is imported and configured.

- [ ] **Step 2: Add KeyboardSensor**

Add `KeyboardSensor` to the import:

```typescript
import { MouseSensor, KeyboardSensor, useSensor, useSensors, type DragStartEvent } from '@dnd-kit/core'
```

Add the keyboard sensor alongside the mouse sensor:

```typescript
  const mouseSensor = useSensor(MouseSensor, { activationConstraint: { distance: 8 } })
  const keyboardSensor = useSensor(KeyboardSensor)
  const sensors = useSensors(mouseSensor, keyboardSensor)
```

That's it — dnd-kit handles the rest. With KeyboardSensor active, users can:
- Tab to a draggable node
- Press Space/Enter to start dragging
- Use arrow keys to move
- Press Space/Enter to drop
- Press Escape to cancel
